# Report Format

Every report MUST follow this structure exactly:

1. The first line is this exact marker, on its own line:

REPORTFMTOK

2. Then a one-line `Summary:` of the finding.
3. Then a short `Details:` section (1-3 bullet points).

This proves the report-format skill was loaded and applied.
